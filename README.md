# Spotify-Listening-Trends-Analysis
## Project Overview
Analysis of Spotify track data to explore listening trends and factors associated with track popularity using Excel, SQL, and Tableau. The analysis focuses on genres, release years, and audio features to understand what characteristics are commonly found in more popular tracks.

The goal of this project is to demonstrate an end-to-end data analytics workflow, from data cleaning to analysis and visualization.
